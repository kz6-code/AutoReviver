# AutoReviver
UnihackInnovationFest Project 3

AutoReviver combines VIN decoding, fitment verification, and AI-powered part recognition into one platform.

Core Features
VIN Decoder

Users enter a VIN number to instantly retrieve vehicle information including:

Manufacturer
Model
Year
Engine details
Trim and specifications

This creates an accurate vehicle profile used throughout the platform.

Smart Part Search

After decoding a VIN, users can search for parts specifically compatible with their vehicle.

This reduces:

Incorrect purchases
Returns
Search time
Image-Based Part Recognition (Prototype)

Users can upload or take photos of car parts to:

Identify unknown parts
Match similar listings
Improve seller uploads
Assist buyers searching visually
Seller Assistance Tools

Prototype tooling helps scrapyards and dismantlers:

Categorise parts faster
Improve listing quality
Reach more buyers online

Tech Stack:

Frontend
HTML / CSS / JavaScript
Responsive web interface

Backend
VIN decoding APIs
Fitment logic and filtering
AI/image matching prototypes
APIs & Data
NHTSA vPIC API for VIN decoding
Vehicle compatibility datasets

How It Works
User enters a VIN number
Vehicle information is decoded automatically
Compatible parts are filtered and displayed
Users can optionally upload images for part identification
Sellers can upload parts with AI-assisted matching
